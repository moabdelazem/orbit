# orbit

## Description
A Go project created with modern project structure.

## Project Structure
- cmd/: Main applications
- internal/: Private application code
- pkg/: Library code
- docs/: Documentation
- test/: Tests

## Getting Started
1. Build the project:
   ~~~
   go build ./cmd/...
   ~~~

2. Run the application:
   ~~~
   go run ./cmd/main.go
   ~~~
